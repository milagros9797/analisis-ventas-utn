# analisis-ventas-utn
TP Organización Empresarial - Análisis de Ventas
## Integrantes
- Milagros Herrera (Trabajo individual - Roles: Hugo, Paco y Luis)

## Escenario
Escenario B – Análisis de Ventas de una Pequeña Empresa

## Dataset utilizado
Dataset simulado de ventas comerciales con registros de producto, cantidad, precio y fecha de venta. Almacenado en /datos/ventas.csv

## Descripción del proyecto
El proyecto trabaja con un dataset de ventas para calcular algunos indicadores básicos: cuánto se vendió en total, qué producto tuvo más salida y cómo fueron evolucionando las ventas mes a mes. Con esos datos se genera un gráfico de barras para visualizarlo de forma más clara.

## Estructura del repositorio
- /datos → dataset de ventas en CSV
- /scripts → script de análisis en Python
- /resultados → gráfico generado

## Cómo ejecutar el script
1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar el script analisis_ventas.py desde la carpeta /scripts
